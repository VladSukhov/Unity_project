# Прототип разрушаемой среды в Unity
Выполнил Сухов В.А. гр. 401Б-22

Минимальный прототип на Unity, демонстрирующий **префрагментированное разрушение** и **каскадное обрушение конструкций**. Фокус на физической симуляции разрушений и логике структурной целостности, а не на полноценном геймплее.

## Возможности
- **Префрагментированное разрушение**: Объекты разделены на физические сегменты, соединённые джойнтами. Урон разрывает связи и применяет импульсные силы.
- **Система структурной целостности**: Граф зависимостей отслеживает опорные элементы. Удаление критической опоры запускает каскадное обрушение зависимых сегментов.
- **Оптимизация физики**: Непрерывная детекция коллизий, настроенные итерации солвера, пороги "сна" и матрица коллизий по слоям для предотвращения туннелирования и дрожания.
- **Визуал на URP**: Кастомные прозрачные стеклянные стены, матовые/бетонные материалы и студийное освещение.
- **Облетающая камера**: Авто-вращение с ручным управлением через ПКМ для кинематографичного просмотра.
- **Эффекты пыли**: Система частиц, создается при разрушении для визуальной обратной связи.

## Управление
- `ЛКМ` — Нанести урон / разрушить структурный элемент
- `ПКМ` + Движение мыши — Ручное вращение камеры
- *(Авто-вращение включено по умолчанию)*

## Примечания
- Проект разработан как **технический прототип**, а не полноценная игра
- Оптимизирован для ~50-70 одновременных фрагментов; используется переключение kinematic/dynamic и оптимизация "сна"
- Без сетевого кода и постоянного состояния; вся симуляция локальная и физическая
- Граф структурных связей строится во время выполнения; опоры можно размещать вручную или через префабы
